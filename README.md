# 🔧 Sistema Bancada TI

Sistema de gerenciamento de chamados para assistência técnica.

## 🛠️ Como foi feito

- **Backend:** Python com Flask
- **Banco de dados:** MySQL (XAMPP)
- **Frontend:** HTML, CSS, JavaScript
- **Bibliotecas:** Bootstrap, jQuery, Select2, DataTables, Chart.js

## 📋 O que tem no sistema

- ✅ **Login** - Autenticação de usuários
- ✅ **Kanban** - Quadro com colunas (Entrada, Manutenção, Pronto, Entregue)
- ✅ **Drag and drop** - Arrastar cards entre colunas
- ✅ **Cadastro de chamados** - Equipamentos com defeito
- ✅ **Cadastro de usuários** - Técnicos do sistema
- ✅ **Lista de chamados** - Tabela com busca e filtros
- ✅ **Relatórios** - Gráficos e exportação PDF/Excel
- ✅ **Logs** - Registro de todas as ações
- ✅ **Tempo real** - Atualizações com Socket.IO
